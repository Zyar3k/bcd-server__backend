# Betty Craft Designs - server

### [LIVE DEMO](https://craftshopdb-server.onrender.com/)

> Online shop server connected with mongoDB

#### Backend

##### Used in project

- cors
- express
- mongoose
- nodemon

##### Available endpoints:

```sh
- /api/products - all products
```

```sh
- /api/product/:id - products by _id
```

### Installation

To run app

```sh
npm i / yarn install
npm start / yarn start
```
